Reinforcement Learning
6th Semester Reinforcement Learning Lab
📌 Overview

This repository contains practical implementations and lab experiments for the Reinforcement Learning (RL) course (6th Semester).
The main objective of this lab is to understand and implement core RL concepts using Python.

Reinforcement Learning is a branch of Machine Learning where an agent learns to make decisions by interacting with an environment and receiving rewards or penalties.

🎯 Objectives

Understand the basics of Reinforcement Learning

Implement Markov Decision Process (MDP)

Apply Dynamic Programming methods

Implement Policy Iteration and Value Iteration

Understand Q-Learning and basic RL algorithms

Analyze agent behavior using rewards and discount factors
